# sky-homepage
---
## introduction
I will make my homepage with web, desktop app, and mobile  

## commands
1. run web server
```bash
npm run web:start
```
2. run development backend
```bash
npm run web:backend:dev
```
3. run development frontend
```bash
npm run web:frontend:dev
```
4. build frontend
```bash
npm run web:frontend:build
```
